Pedestria

Pedestria is an online Minecraft server, which allows Java and Bedrock Edition players to play on one of two sub-servers, with more planned to release in the future. The two current options are Survival and WarLands. This server was only available to Java players from 10/14/2019, until 7/20/2020.
